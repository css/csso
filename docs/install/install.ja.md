# npmでのインストール

事前に必要なもの:

* nodejs 0.4.x&nbsp;— [http://nodejs.org](http://nodejs.org)
* npm&nbsp;— [http://github.com/isaacs/npm/](http://github.com/isaacs/npm/)

インストール（グローバル）方法:

* `npm install csso -g`

アップデート方法:

* `npm update csso`

アンインストール方法:

* `npm uninstall csso`

# gitでのインストール

事前に必要なもの:

* git&nbsp;— [http://git-scm.com/](http://git-scm.com/)

インストール法:

* `git clone git://github.com/css/csso.git`
